# 👔 HR-Ассистент для Новичков

Специализированный модуль для помощи новым сотрудникам в адаптации и ответах на HR-вопросы через Telegram.

## 📋 Описание

HR-ассистент - это дополнительный модуль к основному боту, который:
- Отвечает на вопросы новых сотрудников о компании
- Помогает с онбордингом и адаптацией
- Использует базу знаний (RAG) для точных ответов
- Автоматически определяет HR-вопросы

## 🚀 Возможности

### Основные функции

1. **Ответы на HR-вопросы**
   - Оформление документов (отпуск, больничный, заявления)
   - Информация о зарплате и премиях
   - График работы и правила
   - Процесс онбординга

2. **Онбординг новичков**
   - Помощь в первом дне
   - Знакомство с правилами компании
   - Объяснение процессов

3. **Интеграция с базой знаний**
   - Ответы на основе документов компании
   - Указание источников информации
   - Контекстно-зависимые ответы

## 📱 Использование

### Команды

```
/hr - Показать справку по HR-ассистенту
/hr/reset - Сбросить контекст диалога
/hr/status - Показать статус онбординга
/mode hr - Переключиться в HR-режим
```

### Примеры вопросов

```
• Как оформить отпуск?
• К кому обратиться за помощью?
• Как работает система оценки?
• Где найти правила компании?
• Какой график работы?
• Как получить справку?
```

### Автоматическое определение

HR-ассистент автоматически реагирует на вопросы, содержащие ключевые слова:
- отпуск, больничный, зарплата, премия
- оформление, договор, кадров
- график, опоздание, увольнение
- руководитель, команда, коллеги
- правила, политика, регламент
- новичок, онбординг, адаптация

## ⚙️ Архитектура

### Компоненты

```
services/hr_assistant.py     # Специализированный HR-клиент
handlers/hr.py               # Обработчики HR-команд
services/router.py           # Маршрутизация HR-запросов
utils/helpers.py             # Управление сессиями онбординга
```

### Поток обработки

```
1. Пользователь задаёт вопрос
   ↓
2. Автоматическое определение HR-тематики
   ↓
3. Поиск в базе знаний (RAG)
   ↓
4. Генерация ответа с контекстом
   ↓
5. Отправка ответа через Telegram
```

## 🔧 Настройка

### Добавление в проект

1. Убедитесь, что файлы созданы:
   - `services/hr_assistant.py`
   - `handlers/hr.py`
   - Обновлён `services/router.py`
   - Обновлён `config.py` (добавлен `BotMode.HR`)

2. Обновите `handlers/__init__.py`:
   ```python
   from handlers import hr
   ```

3. Добавьте HR-документы в базу знаний:
   ```bash
   # Поместите документы в data/documents/
   # Они автоматически будут индексируются RAG
   ```

### База знаний

Добавьте в `data/documents/` следующие документы для лучшего результата:

```
data/documents/
├── employee_handbook.pdf       # Руководство сотрудника
├── hr_policies.pdf             # HR-политики
├── onboarding_guide.pdf        # Гайд по онбордингу
├── benefits.pdf                # Льготы и бонусы
├── procedures.pdf              # Процедуры и процессы
└── faq_hr.pdf                  # Частые вопросы HR
```

## 🎯 Интеграция с RAG

HR-ассистент использует существующую RAG-инфраструктуру:

```python
# Поиск в базе знаний
results = rag_query.similarity_search_with_score(query, k=3)

# Если найдены документы - используем контекст
if results:
    context = prepare_context(results)
    response = await hr_assistant.hr_consultation(
        query=query,
        context=context
    )
```
