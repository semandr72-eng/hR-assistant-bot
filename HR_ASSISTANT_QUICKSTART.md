# 🚀 Быстрый старт: HR-Ассистент

## ✅ Что создано

HR-ассистент для помощи новичкам в компании интегрирован в существующий Telegram-бот.

### Созданные файлы

```
services/hr_assistant.py          # Специализированный HR-клиент
handlers/hr.py                     # Обработчики HR-команд
tests/test_hr.py                   # Тесты (9 тестов, все проходят)
docs/HR_ASSISTANT.md              # Полная документация
.env.example                       # Пример конфигурации
```

### Обновлённые файлы

```
config.py                          # Добавлен BotMode.HR
services/router.py                 # Добавлена маршрутизация HR-запросов
handlers/text.py                   # Добавлен HR-режим в список
utils/helpers.py                   # Добавлено управление онбордингом
handlers/__init__.py              # Импорт HR-модуля
```

## 📋 Использование

### 1. Настройка окружения

```bash
# Скопируйте пример конфигурации
cp .env.example .env

# Отредактируйте .env и добавьте ваши ключи:
TELEGRAM_BOT_TOKEN=your_token_here
OPENAI_API_KEY=your_api_key_here
```

### 2. Добавление базы знаний

Поместите HR-документы компании в директорию `data/documents/`:

```
data/documents/
├── employee_handbook.pdf       # Руководство сотрудника
├── hr_policies.pdf             # HR-политики
├── onboarding_guide.pdf        # Гайд по онбордингу
├── benefits.pdf                # Льготы и бонусы
└── procedures.pdf              # Процедуры и процессы
```

### 3. Запуск бота

```bash
python main.py
```

### 4. Работа с HR-ассистентом

#### Вариант А: Автоматическое определение
Просто задайте HR-вопрос в чате:
```
/.start
Как оформить отпуск?
Где найти правила компании?
```

#### Вариант Б: Явный HR-режим
```
/mode hr
Сколько дней отпуска?
Каков график работы?
```

#### Команды HR-ассистента
```
/hr                 - Показать справку
/hr/reset           - Сбросить контекст
/hr/status          - Показать статус онбординга
```

## 📚 Возможности

### Ответы на вопросы
- Оформление документов (отпуск, больничный, заявления)
- Информация о зарплате и премиях
- График работы и правила
- Процесс онбординга

### Интеграция с RAG
- Ответы на основе документов компании
- Указание источников информации
- Контекстно-зависимые ответы

### Отслеживание онбординга
- Статус адаптации новичка
- История вопросов
- Прогресс обучения

## 🔧 Архитектура

```
Пользователь задаёт вопрос
         ↓
Автоматическое определение HR-тематики

         ↓
Отправка ответа через Telegram
```
